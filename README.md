# ⏱️ Focus Time

A minimal and modern **study focus tracker** designed to help you stay productive and track your daily focus sessions.

🌍 **Live Demo**
https://focus-time-roan.vercel.app

---

## 🚀 Features

* ⏱️ Focus Timer (track study time)
* 🔐 User Authentication (Supabase)
* 📊 Study Analytics (charts & stats)
* 🎯 Daily goal tracking
* ⚡ Fast and responsive UI

---

## 🧠 How to Use

1. Visit the live app
2. Sign up or log in
3. Start a focus session
4. Track your progress and stay consistent

---

## 🛠️ Tech Stack

  ⚛️ React (Vite)

  🟢 Supabase (Auth + Database)

  📊 Chart.js

  ▲ Vercel (Hosting)
